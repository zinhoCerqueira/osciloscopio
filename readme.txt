###Instruções para executar e como funciona o programa

- Desenvolvido em pythoh, usando as bibliotecas Mathplot;
- É necessário instalar as bibliotecas numpy, mathplotlib, random, drawnow

#Funcionamento
O sinal é gerado usando a biblioteca random, que gera valores aleatórios. Esses valores são armazenados em um array, de tamanho 256, que é a quantidade necessária para aplicar a FFT (Transformada de Fourier).

Esse sinal, após tratado, é apresentado no gráfico.
